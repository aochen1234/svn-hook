项目部署说明:
1. 利用nginx和uwsgi部署命令
a.nginx:
fnngj@ubuntu:~$ /etc/init.d/nginx start  #启动
fnngj@ubuntu:~$ /etc/init.d/nginx stop  #关闭
fnngj@ubuntu:~$ /etc/init.d/nginx restart  #重启

b.uwsgi:
sudo uwsgi --ini uwsgi 启动命令
sudo fuser -k 8080/tcp  杀掉占用端口

2.配置
a.在项目目录下创建uwsgi.ini
[uwsgi]

# Django-related settings

http-socket = :8080

# the base directory (full path)
chdir           = /home/ub/webshell/testapp

# Django s wsgi file
module          = testapp.wsgi

# process-related settings
# master
master          = true

# maximum number of worker processes
processes       = 4

# ... with appropriate permissions - may be needed
# chmod-socket    = 664
# clear environment on exit
vacuum          = true

b.vi /etc/nginx/sites-enabled/default 文件
server {
        listen 8080;

        server_name 192.168.117.100;
        charset  utf-8;
        access_log   /var/log/nginx/myweb_access.log;
        error_log       /var/log/nginx/myweb_error.log;
        client_max_body_size  75M;
        location /static {
            alias /home/ub/webshell/testapp/static;
        }
        location /{
            include    uwsgi_params;
            uwsgi_pass 127.0.0.1:8080;
            uwsgi_read_timeout 2;
        }
